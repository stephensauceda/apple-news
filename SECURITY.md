# Security Policy

## Supported Versions

Only the latest published version is guaranteed to receive security updates.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead, report privately by emailing:

- contact@stephensauceda.com

Include as much detail as possible:

- Affected version
- Reproduction steps
- Potential impact
- Any suggested remediation

## Response Expectations

- Initial response target: within 5 business days
- Status updates: as fixes are developed and validated
- Coordinated disclosure: after a fix is released
